EEG Neural Signal Classifier | Python, MNE, scikit-learn | [github]
Built an end-to-end EEG processing pipeline using MNE
preprocessing (bandpass filter, notch filter, ICA artifact removal), 
epoch extraction, and spectral feature engineering (delta/theta/alpha/beta band power). 
Trained an LDA classifier with 5-fold cross-validation for auditory stimulus classification.
